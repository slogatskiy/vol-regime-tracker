Method & sources

Data honesty

Live series (MOVE, data-day vol, dealer balance sheet) refresh from public APIs on every collector run — each refresh is a git commit, so history is diffable. Illustrative series are flagged amber: the shape is seeded for design and is overwritten when the sourced reports (MKTX/TW IR, SIFMA, CME) are ingested. Nothing illustrative is presented as a measured value.
